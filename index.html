Principais palavras-chaves e comandos: Palavras-chave de v1.0

Formas

Objetos 2d para fazer artes ou animações. Sintaxe: forma: >> Suas propriedades aqui...

- circle: Cria um objeto redondo, útil para fazer olhos de um personagem.

- square: Cria um quadrado na tela.

- triangle: Cria um triangulo.

- star: Cria uma estrela.

- line: Cria uma linha.

- text = "Seu texto": Cria um texto.

Propriedades

- size: O tamanho do seu objeto. Funciona para todos exceto linha.

size = 50px (padrão) | size = 30px 20px (largura/altura)

- position: Posição do objeto (centro = 0px 0px).

position = 15px 20px | position = center | Linhas: position = 15px 30px, 20px 15px

- color: Cor (nome em inglês ou Hexadecimal).

color = red | color = #FF0000

- thickness: Espessura da linha (apenas para line).

thickness = 15px | thickness = standard (padrão 12px)

- zindex: Define profundidade (frente/atrás).

zindex = standard (0) | zindex = 3

- >>: Comentário no código.

>> Boo!

Suporte de Propriedades

  • Objetos comuns: size, position, color, zindex
  • Texto: size, position, color, zindex, thickness
  • Linha: position, color, zindex, thickness

Novas palavras-chave e comandos: Palavras-chave de v1.1

varSelector

Banco de dados do 2&3Dix para escrever variáveis.

varSelector:
  nome = "Yudhi"
  altura = 30px

nome = valor

Definição de variável. Valor pode ser texto, número ou pixels.

Cálculos permitidos: diametro = raio * 2 | pixel = pixel + 1

id

Identidade do objeto.

circle (id = "cabeca"):
  color = #FFC080

class

Agrupamento de objetos.

circle (class = "objetos"):
  color = red

n-gons

Polígono personalizado (substitua n pelo número de lados).

10-gon:
  color = red >> Um decágono.

print ->

Manda mensagem ao console (não desenha).

print -> "Estou no console, e concluí a v1.1!"
