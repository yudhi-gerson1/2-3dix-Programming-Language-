Principais palavras-chave e comandos:

2&3Dix: Comandos 2D v1.0

Formas

Objetos 2d para fazer artes ou animações.

Exemplo de Sintaxe:
forma: 
  >> Suas propriedades aqui...

- circle

Cria um objeto redondo, útil para fazer olhos de um personagem.

circle:
  size = 50px
  color = black

- square

Cria um quadrado na tela.

square:
  size = 50px
  color = red

- triangle

Cria um triangulo.

triangle:
  size = 50px
  color = blue

- star

Cria uma estrela.

star:
  size = 50px
  color = yellow

- line

Cria uma linha.

line:
  position = 0px 0px, 50px 50px
  thickness = standard
  color = green

- text = "Seu texto"

Cria um texto.

text = "Seu texto":
  position = center
  color = black

Propriedades: A definição do seu objeto

Exemplo de Sintaxe:

square:
  propriedade = tipo

- size

O tamanho do seu objeto. Funciona para todos os objetos menos o line.

Exemplo 1: size = 50px (o padrão)
Exemplo 2: size = 30px 20px (30px de largura, 20px de altura)
Exemplo 3: size = standard (o padrão é 50px)

square:
  size = 30px 20px

- position

A posição do seu objeto. O centro é 0px 0px.

Exemplo 1: position = 15px 20px (15px para direita, 20px para cima)
Exemplo 2: position = center (0px 0px)
Para as linhas: Tem que definir a posição de dois pontos, fazemos assim:

position = 15px 30px, 20px 15px

- color

A cor do seu objeto. Pode ser com as palavras reservadas (red, blue, etc.) ou com hexadecimal.

Exemplo 1: color = red
Exemplo 2: color = #FF0000

circle:
  color = #FF0000

- thickness

A espessura da linha (só pode ser usado para line).

Exemplo 1: thickness = 15px.
Exemplo 2: thickness = standard (o padrão é 12px).

line:
  thickness = 15px

- zindex

Define se o objeto está na frente ou atrás.

Exemplo 1: zindex = standard (Define o zindex como 0)
Exemplo 2: zindex = 3

square:
  zindex = 3

Novas palavras-chave e comandos:

2&3Dix v1.1 2D: Comandos novos e Palavras-chave.

varSelector

O banco de dados do 2&3dix, usado para escrever variáveis.

varSelector:
  nome = "Yudhi"
  altura = 30px

text = nome:
  position = 0px altura

nome = valor

A definição de uma variável. O valor pode ser um texto, um número inteiro ou pixeis (de uma a duas dimensões). Se for um número, permite fazer cálculos. E também pode ser atualizado.

Exemplo: largura = 30px (obrigatoriamente dentro de varSelector)

diametro = raio * 2
pixel = pixel + 1

id = meu_objeto

A identidade do objeto.

circle (id = "cabeca"):
  color = #FFC080

class = meus_objetos

A classe dos objetos.

Exemplo 1:

circle (class = "objetos"):
  color = red

square (class = "objetos")
  color = red

Exemplo 2:

circle (class = "objetos")
square (class = "objetos")

objetos:
  color = red

n-gons

Um polígono personalizado, pode substituir n por qualquer número.

10-gon:
  color = red
  >> Um decágono.

print -> "texto"

Quando você quer mandar a mensagem no console, não no desenho.

square:
  color = red

print -> "Estou no console, e concluí a v1.1 de 2&3dix!"

Lazy definition

Explicação Detalhada: Ocorre quando você declara e define a existência dos objetos pertencentes a uma determinada classe primeiro no arquivo, deixando para especificar as propriedades e estilizações dessa classe apenas em um bloco posterior do código.

square (class = "blue")
circle (class = "blue")

blue:
  color = blue

Herança

Explicação Detalhada: Ocorre quando você inverte a ordem sequencial de leitura: você cria e define completamente as propriedades da classe no topo do escopo, antes mesmo de instanciar ou declarar os objetos que farão uso dessa classe.

blue:
  color = blue

square (class = "blue")
circle (class = "blue")

Filho Independente

Explicação Detalhada: Acontece quando um objeto específico vinculado a uma classe herda e carrega automaticamente todas as propriedades globais daquela classe, mas possui a flexibilidade única de mesclar, estender ou sobrescrever esses valores com as suas próprias propriedades locais individuais.

blue:
  color = blue

square (class = "blue"):
  zindex = 2
circle (class = "blue")
