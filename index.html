Documentação Geral: Comandos 2D

Formas

Objetos 2D prontos para construir suas artes ou animações no canvas.

forma:
  >> Suas propriedades aqui...
  • circle

    Cria um objeto redondo, útil para fazer olhos de um personagem.

  • square

    Cria um quadrado na tela.

  • triangle

    Cria um triângulo.

  • star

    Cria uma estrela.

  • line

    Cria uma linha reta conectando pontos.

  • text = "Seu texto"

    Escreve uma cadeia de caracteres na tela.

Propriedades

Definições e customizações aplicadas a cada tipo de objeto inserido.

square:
  propriedade = tipo
  • size

    Define o tamanho do objeto (não funciona para line).

    size = 50px (padrão) size = 30px 20px (Largura / Altura) size = standard
  • position

    Define as coordenadas espaciais. O centro do canvas é sempre 0px 0px.

    position = 15px 20px (Direta / Cima) position = center position = 15px 30px, 20px 15px (Apenas p/ Linhas)
  • color

    Altera a cor do traço/preenchimento usando nomes reservados ou Hexadecimal.

    color = red color = #FF0000
  • thickness

    A espessura da linha ou contorno (Exclusivo para line e text).

    thickness = 15px thickness = standard (padrão é 12px)
  • zindex

    Gerencia as camadas visuais, decidindo quem renderiza na frente ou atrás.

    zindex = standard (padrão é 0) zindex = 3
